# 数组和指针
在Go语言里面，数组是一个基本类型，所以数组赋值和传参都会复制整个
数组，如果不想这样，就需要使用指针，通过&获取地址。
a := [2]int {10, 20}
b :=a  /* b 和 a 都有一份【10,20】的拷贝*/
c :=&a /* c 和 a 共用一份数据 */
